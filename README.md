# How to install
1. Head over to `chrome://extensions/` (or `edge://extensions/` if that's your thing).
2. Toggle "Developer mode" on (usually in the top right corner).
3. Click "Load unpacked" and just select this entire folder.
4. You're good to go.

# How to use
Whenever you open a Google Form, a little overlay will pop up in the bottom right corner. 
Just type in your details (name, ADDU email, section), hit save, and smash that autofill button whenever you need to fill out a form. 

Saves time-ish.
