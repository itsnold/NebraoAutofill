# How to install
1. Download the zip.
2. Right-click the downloaded `.zip` and hit "Extract to Nebrao_Autofill\" (just make sure it extracts into its own folder so files don't spill everywhere).
3. Head over to `chrome://extensions/` (or `edge://extensions/` 🫠).
4. Toggle "Developer mode" on (usually in the top right corner).
5. Click "Load unpacked" and just select the folder you just extracted.
6. You're good to go.
